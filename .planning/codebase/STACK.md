# Codebase Stack

## Core Technologies
- **Language**: PHP 8.2+
- **Framework**: Laravel 12.x
- **Database**: MySQL 8.0+ (Local)
- **Frontend Bundler**: Vite 7.x
- **CSS Framework**: Tailwind CSS 4.x

## Key Packages
- **Laravel Boost**: 2.x (Agentic coordination)
- **Vite Plugin**: laravel-vite-plugin 2.x
- **Tailwind Plugin**: @tailwindcss/vite 4.x
- **HTTP Client**: Axios

## Development Environment
- **Server**: PHP Built-in Server (php artisan serve)
- **Tooling**: Laravel Pint (linting), PHPUnit (testing)
