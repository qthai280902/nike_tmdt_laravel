# Phase 2 Plan: Product Ecosystem & Seeders

## Backend: Data Seeding (Foundation)
- **CategorySeeder**: Create a hierarchical tree (Men -> Lifestyle, Running; Women -> Training, etc.).
- **ProductSeeder**: Generate ~12 flagship products with unique slugs and high-quality image URLs.
- **VariantSeeder**: Map standard Nike sizes (US 7-12) and colorways to each product.
- **Factory Integration**: Use Laravel Factories for variants to ensure mass-safety and unique SKUs.

## Backend: Service Expansion
- **ProductService**:
  - Add `filterCatalog($filters)` method.
  - Implement `$query->whereHas('category', ...)` for category filtering.
  - Implement `$query->whereHas('variants', ...)` for size/color filtering.
  - Implement `$query->orderBy()` based on price sort parameters.
- **ProductController**:
  - Update `index` to accept query parameters (`category`, `size`, `color`, `sort`).

## Frontend: Nike Radical Minimalist (Grid & Detail)
- **PLP (Product Listing Page)**:
  - 3-column CSS Grid (Gap: 12px) for desktop.
  - Minimal product cards: Borderless, square image, metadata below.
  - Dynamic category/filter sidebar (or top bar).
- **PDP (Product Detail Page)**:
  - Full-bleed vertical image gallery.
  - Sticky info panel on desktop.
  - Variant selection pills (sizes/colors) with "Unavailable" state (grey-scale).
  - Kinetic "Add to Bag" button (Pill, Primary).
